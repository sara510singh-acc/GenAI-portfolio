# TECH TUTOR

An AI-powered interactive learning platform designed for Computer Science students.
This application helps users practice SQL, understand AI/ML concepts visually, and test their knowledge through quizzes — all with intelligent feedback.

---

##  Features

###  1. English → SQL Converter + AI Feedback

* Users input a query in plain English
* System converts it into SQL
* AI evaluates the generated SQL and provides:

  * Correctness feedback
  * Optimization suggestions
  * Explanation of the query

 Example:

* Input: "Get all students with marks greater than 80"
* Output: SQL query + AI feedback

---

###  2. AI/ML Image Generation + Explanation

* Generate images based on AI/ML concepts

* Example topics:

  * Neural Networks
  * Decision Trees
  * Data Pipelines

* Users can:

  * View generated image
  * Speak or explain the concept for ~30 seconds

* AI evaluates:

  * Concept clarity
  * Explanation quality

---

###  3. Smart Quiz Generator (MCQ / Fill in the Blanks)

* Generates questions based on:

  * Data Science
  * AI/ML topics

* Adjustable difficulty:

  * Easy / Medium / Hard

* AI evaluates responses:

  * Correctness
  * Explanation
  * Improvement suggestions
